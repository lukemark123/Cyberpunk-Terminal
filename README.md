# Cyberpunk Terminal

A neon, cyberpunk-inspired terminal interface built in Python using [Textual](https://github.com/Textualize/textual) and [Rich](https://github.com/willmcgugan/rich).  
Designed as a **portfolio showcase** with retro CRT aesthetics, dynamic prompts, and command history.

![Cyberpunk Terminal Screenshot](screenshot.png)

## Features

- Retro-futuristic terminal UI with neon colors
- Animated login sequence with ASCII glitch art
- Dynamic command prompt with current working directory
- Command history navigation using ↑ / ↓ keys
- Clear terminal with `Ctrl+L`
- Rich outputs: supports tables, panels, and text formatting
